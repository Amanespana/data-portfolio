Comprehensive Analysis of Global CO2 Emissions (Excel)

Objective

To conduct a comprehensive analysis of global CO2 emissions spanning from 1970 to 2021 and answer following key questions:

  • The annual percent change from 1970 to 2021.
  • Trend of global emissions during same period.
  • Identify top 5 countries with highest emission in any given year?
  • The percentage breakdown of CO2 emission from Coal, Oil and Gas?

Process

  1. Data Collection: Kaggle
  2. Data Understanding and Selection: The original dataset contains data from 1750 to 2021. However, for this project, I only required data from 1970 to 2021.
  3. Data Cleaning and Modelling:
    • Check duplicates and missing values to ensure data accuracy
    • Format correct data type for each column
    • Create Pivot tables, calculated fields, and slicers
  4. Data Visualization:
    • Create a column chart for annual percent change from 1970 to 2021
    • Create a line chart for global emission trend during the same period
    • Create a column chart to identify the top 5 countries with the highest emission in any given year
    • Create doughnut charts for the breakdown of CO2 emission from Coal, Oil, and Gas
  5. Data Storytelling: Dashboard and Insights
    • Bring everything together on a single interface to tell the story
    • Insights
      • In 2021, global CO2 emissions were 36.1 Gigatonnes. Out of this, 41% was from coal, 30% from oil, and 22% from gas
      • Top 5 Countries with highest emissions are China, USA, India, Russia, Japan, in descending order
      • On average there has been 1.83% increase in global CO2 emissions per year from 1970 to 2021.
